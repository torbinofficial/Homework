If u wanna recreate project:
	Create Java Maven project
	In pow.xml write dependencies for Cucumber, jUnit and Selenium, Eclipse will install them
	Then download chromedriver.exe and put into /src/main/resources/drivers/
	Put Feature files into /src/test/resources/Features/
	Then write realisation in /src/test/resources/java/

If u wanna just download and test:
	Clone git anywhere u want
	Run tests from .feature file as 'Cucumber Feature'
